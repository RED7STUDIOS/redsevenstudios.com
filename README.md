# redsevenstudios.com

This is the source code for [redsevenstudios.com](https://redsevenstudios.com).

## Development

We use [bun](https://bun.sh).

1. Install dependencies:

   ```bash
   bun i
   ```

2. Run the server:

   ```bash
   bun dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)
